# RetroSort
A Python script to sort and transfer local rom files to a remote RetroPie installation.

## Requirements
* RetroPie installed on a Raspberry Pi with SSH enabled
* Python 3 - this script has not been fully tested on Python 2
* [Python pip](https://pip.pypa.io/en/stable/installing/)

## Installation

Using git:
```
cd
git clone https://github.com/rycolos/RetroSort.git
cd RetroSort
```
Using wget:
```
cd
wget https://github.com/rycolos/RetroSort/archive/master.zip
unzip master.zip
cd RetroSort-master
```
Install dependencies:
```
pip install -r requirements.txt
```
